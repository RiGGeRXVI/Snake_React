# Snake Game
Онлайн игра "Змейка" на React + TypeScript (клиент) и Node.js + Express + Socket.IO + TypeScript (сервер)

## Что реализовано на сайте
- Home: ввод и подтверждение ника игрока (сохранение в localStorage)
- Customization: выбор цвета змейки (сохранение в localStorage), превью змейки.
- Game:
    - Подключение к серверу и постановка в очередь;
    - Матч стартует, когда в очереди 2 игрока;
    - Управление змейкой через WASD(без поддержки русской раскладки) или стрелки;
    - Сервер рассылает состояние игры (две змейки + фрукт)
    - При завершении матча показывается результат и кнопки:
        - Сыграть снова
        - Перейти к результатам
- Leaderboard:
    - Сервер хранит статистику (wins / games)
    - Клиент показывает таблицу лидеров (топ-20) и кнопку обновления.

## Требования
- Node.js
- npm

## Устновка
В корне проекта ввести ```npm install```

## Запуск проекта
1) Запуск сервера (Socket.IO + API)
В терминале ввести:
```npm -w server run dev```

По умолчанию сервер стартует на: ```http://localhost:3001```

2) Запуск клиента (Vite)
В отдельном терминале ввести:
```npm -w client run dev```
По умолчанию клиент стартует на: ```http://localhost:5173```

# Как играть
1. Откройте сайт (клиент)
2. На домашней странице введите ник и нажмите "Подтвердить"
3. При желании выбрать цвет в Customization
4. Перейдите на Game - вы попадёте в очередь
5. Открыть второй браузер или вкладку в режиме инкогнито, введите другой ник и тоже зайдите в Game - матч начнётся автоматически
6. Управление: WASD / стрелки (русская раскладка не поддерживается!)
7. После окончания матча можно сыграть снова или открыть таблицу лидеров.
