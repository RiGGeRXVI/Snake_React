Браузерная игра "Змейка"

# Lab-7

## Запуск проекта
1. Установка зависимостей
```cd client```
```npm install```
2. Запуск в режиме разработки
```npm run dev```
3. Перейти по адресу ```http://localhost:5173```

## Тестирование
Запуск тестов:
```npm run test```

Генерация покрытия:
```npm run test:coverage```

### Линтинг
Настроен строгий ESLint с правилами:
2 пробела
Без ;
Без пустых строк
Запрет any
Явные типы функций
Строгая типизация

Проверить ошибки:
```npm run lint```

Автоматическое исправление:
```npm run lint:fix```

## Роутинг
Приложение включает 4 страницы:
Главная (HomePage) — описание проекта
Игра (GamePage) — заглушка игрового поля
Таблица лидеров (LeaderboardPage) — заглушка таблицы
Кастомизация (CustomizationPage) — заглушка настроек змейки
Все страницы подключены в AppRouter.tsx.
